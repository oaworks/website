Major Updates for the Open Access Button and Open Data Button

We're excited to say this week we'll release major updates for the Open Access Button and the Open Data Button for Firefox.

Updated plug-ins for the Open Access Button will now have more ways to find access to research. First, we've integrated with with BASE. Base indexes over 5.5 million Open Access articles, primarily from repositories. BASE joins CORE, as services the Open Access Button can now use to find access to articles behind a paywall. Second, the plugin supports our "email the author" and request systems. To support these systems we've integrated with Dissem.in and Zenodo. Papers obtained through the system will be deposited through Dissem.in and Zenodo.

Along with these new features, the map and stories from the Open Access Button beta will return, and both the map and user stories will now be embeddable and searchable.

Finally, we've made several bug fixes within this update, including a better sign-in experience for users and how the plug-in identifies papers on publisher websites.

Both the updated Google Chrome and Mozilla Firefox plug-ins for the Open Access Button will be released this week. This release will improvethe Open Access Button's usability, ability to find access to research for users, and enable users to request access to more research.

Joining this update will be the Open Data Button's release on Firefox.
